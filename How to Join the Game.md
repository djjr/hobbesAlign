# Review the Rules

This activity proceeds in rounds. In each round you get to choose a territory (some are more fecund - more likely to have useful resources you can harvest) and decide what action you will take:
- Cooperate with another agent if you meet them. If there are resources to harvest and you meet another cooperator you both get a bonus.
- Give a unit of resource to another agent you might meet.
- Go solo meaning if you find a resource you will harvest it by yourself, but if anyone else is there, you have an encounter.
- Attempt to rob and kill another agent you might meet.

The outcome of the round depends on what another agent you meet has decided to do. After each round, all agents pay a minimum daily requirement (MDR) now set at 1. If one is completely alone at a node, you harvest 1 if any resources are present, otherwise nothing happens.
